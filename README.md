# tensquare
黑马十次方项目微服务代码

注意：**本代码库仅包含后端微服务代码**

## tensquare介绍

《十次方》是程序员的专属社交平台，包括头条、问答、活动、交友、吐槽、招聘六大频道。

十次方名称的由来：2的10次方为1024，程序员都懂的。
